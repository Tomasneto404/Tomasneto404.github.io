◆ VALOR_INVALIDO

#define VALOR_INVALIDO   "ERROR: Invalid value."

A string constant representing an invalid value message.

This constant is used to define a standard error message for invalid input values. It can be used throughout the program to provide consistent error messages to the user when they enter an invalid value.
